# 🌱 Agri-Food Blog

## 📌 Description

**Agri-Food** est un site web de blog simple dédié à l’agriculture, à l’alimentation et aux innovations dans le domaine agroalimentaire.
Ce projet a été réalisé uniquement avec **HTML et CSS**, sans utilisation de JavaScript.

L’objectif est de créer une interface claire, moderne et accessible pour partager des articles autour de :

* 🌾 L’agriculture durable
* 🍎 L’alimentation saine
* 🌍 Les enjeux environnementaux
* 🚜 Les nouvelles technologies agricoles

---

## 🛠️ Technologies utilisées

* **HTML5** → Structure du site
* **CSS3** → Mise en forme et design

---

## 📂 Structure du projet

```
agri-food/
│── index.html
│── style.css
│── images/
```

* `index.html` : page principale du blog
* `style.css` : fichier de styles
* `images/` : dossier contenant les images du site

---

## 🎨 Fonctionnalités

* 📰 Affichage d’articles de blog
* 🧭 Barre de navigation simple
* 📱 Design responsive (adapté mobile)
* 🎯 Interface claire et lisible

---

## 🚀 Installation et utilisation

1. Télécharger ou cloner le projet
2. Ouvrir le dossier
3. Lancer le fichier `index.html` dans un navigateur

Aucune installation supplémentaire n’est nécessaire.

---

## ✨ Objectif du projet

Ce projet a pour but de :

* Apprendre les bases du développement web
* Comprendre la structure d’un site statique
* Maîtriser le HTML et le CSS

---
